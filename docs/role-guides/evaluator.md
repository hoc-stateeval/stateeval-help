# Evaluator Guide

As an evaluator, you conduct observations, review evidence, and complete evaluations for staff members assigned to you.

## Key Responsibilities

### Observations
- Conducting observations
- Recording observation notes
- Providing feedback

### Evidence Review
- Reviewing YTD evidence
- [Reviewing artifacts](../artifacts/reviewing-artifacts)
- Adding evaluator notes

### Evaluations
- Completing mid-year reviews
- Completing final evaluations
- Assigning scores and ratings

## Your Dashboard

Your evaluator dashboard shows:
- Staff members assigned to you
- Upcoming and overdue tasks
- Evaluation progress for each evaluatee

## Evaluation Workflow

1. **Pre-observation** - Review evaluatee's self-assessment and goals
2. **Observation** - Conduct and document the observation
3. **Post-observation** - Provide feedback and discuss with evaluatee
4. **Evidence collection** - Review artifacts and evidence throughout the year
5. **Mid-year review** - Complete mid-year evaluation (if required)
6. **Final evaluation** - Complete end-of-year evaluation and ratings

## Common Tasks

| Task | Where to Go |
|------|-------------|
| View my evaluatees | Dashboard > My Evaluatees |
| Start an observation | Select evaluatee > Observations > New |
| Review artifacts | Select evaluatee > Artifacts |
| Complete evaluation | Select evaluatee > Final Evaluation |
